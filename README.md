# Data-Management
Stroke Data Analysis
Healthcare analytics has changed specifically in recent years, specifically how we process the patient data. We can use machine learning to process the data and better process the health information. For example, the Stroke Prediction Dataset enables us to solve a common problem such as predicting strokes based on people's profiles and health signs, we will use SAS Studio to analyse the types of data, how we clean it up, where we store it away and how data warehouses help.
